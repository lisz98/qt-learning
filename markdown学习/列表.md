# markdown列表


## 使用*、+、-来进行标记

* 第一项  
* 第二项  
* 第三项

+ 第一项  
+ 第二项  
+ 第三项  

- 第一项  
- 第二项  
- 第三项  

## 有序学列使用数字加上.来表示

1. 第一项
2. 第二项
3. 第三项

## 列表嵌套

列表嵌套只需在子列表中的选项前面添加4个空格即可  

1. 第一项  
    * 第一项嵌套的第一个元素  
    * 第一项嵌套的第二个元素
2. 第二项  
    * 第二项嵌套的第一个元素
